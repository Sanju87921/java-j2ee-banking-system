# Java J2EE Banking System

## Technologies Used
- Java
- J2EE
- Servlets
- JSP
- JDBC
- MySQL
- HTML / CSS

## Features
- Admin Login
- Customer Registration
- Create Bank Account
- Fund Transfer
- Transaction History
- Account Management

## Project Structure
- DAO Layer
- DTO Classes
- Servlets
- JSP Frontend
- MySQL Database

## How to Run
1. Import project into Eclipse
2. Configure MySQL database
3. Deploy on Apache Tomcat
4. Run application in browser
